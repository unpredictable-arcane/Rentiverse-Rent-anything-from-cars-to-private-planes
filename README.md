## Project description

**Rentiverse** is a clean, multi-page site for a vehicle rental concept. It highlights an image-rich **gallery**, a **slideshow/hero**, a **fixed navbar**, a **custom footer**, and optional **booking UI** ‚Äî all responsive and performance-conscious so it‚Äôs great for a portfolio.

The site is intentionally **static** to enable free hosting and simple forking.
